功能
=
* 可自定义按键
* 计时达到3分钟时会生成新的计时器,如果不需要请按 **重置( X )** 按键即可恢复为一个

默认按键
=
* `R` 开始/暂停键
* `X` 重置
* `Esc` 退出按键
